# Smart Meeting Room Allocator

An AI-based system that allocates meeting rooms based on constraints like capacity, time, and facilities.

## Features
- Constraint Satisfaction Problem (CSP) based allocation
- Avoids time conflicts
- Supports priority handling

## Tech Stack
- Java (Backend Logic)
- HTML, CSS, JavaScript (Frontend)

## How to Run
1. Compile Java files
2. Run Main.java
3. Open index.html for UI
